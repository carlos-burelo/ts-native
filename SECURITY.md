# Security Policy

## Supported versions

TSN is currently in alpha. Security fixes are applied to the latest `main` branch.

## Reporting a vulnerability

Please do not open a public issue for undisclosed vulnerabilities.

Report privately with:

- affected component (checker/compiler/vm/cli/lsp),
- reproduction steps or proof of concept,
- impact assessment.

If a private channel is not available yet, open a minimal issue asking for a confidential contact path without disclosing exploit details.
